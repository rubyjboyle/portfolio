Problem

The problem for Market America's Exclusive Brands was brand recognition and knowledge. There was not a centeralized place to learn about these products to build brand awarness and loyalty.

design_services

My Role

My roles for this project were product designer, UX writer, and front end developer. I gathered all the content and assets for each brand then created mockups to go through design and legal approval. I also created reusable code modules for the UI so each page could be built with the same 10 blocks of code.

flag

Goal

The goal was to provide a page of informational interactive content that a user would engage with to learn about the brand. Primarily to then increase sales based on the trust built.

check_circle_outline

Outcome

The outcome was an improvement on the page bounce rate by 50% for several brands; Isotonix, Motives Cosmetics, Ultimate Aloe. Then by 45% for other brands. This lead to an increase in revenue as well by making $10k in the first month for the Isotonix brand.
